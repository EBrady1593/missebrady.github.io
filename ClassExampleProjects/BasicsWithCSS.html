You can make two different types of lists in HTML.

The first type of list is called an unordered list.

  • An unordered list consists of one or more items.
  • Items in an unordered list are not order specific, meaning the order does not matter.

The other type of list is called an ordered list.

  1. Similar to an unordered list, an ordered list consists of one or more items.
  2. Items in an ordered list are in a set order

You can include images in your HTML file.

There are two main ways to include an image.

  1. The first way is to use a link to the picture.
  2. The second way is to download the image and use the path of the image.

You can include links in your HTML file.

Similar to the images, there is two different ways you can do this.

  1. The first way is to use a URL to link the the page. Link to website for the picture above.
